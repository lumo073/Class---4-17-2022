Assignment Operator
        = -> Value Assignment
        x+=y is same as x= x+y
        x-=y is same as x=x-y
        x%=y is same as x=x%y
        x*=y is same as x=x*y
        x/=y is same as x=x/y
        x**=y is same as x=x**y
    
        comparison Operator
        == -> is equal to
        === -> equal value and equal data type
        != -> not equal to
        !== -> not equal value or not equal data type
        <,>,<=,>=

        logical Operator
        or -> ||
        and -> &&
        not -> !
